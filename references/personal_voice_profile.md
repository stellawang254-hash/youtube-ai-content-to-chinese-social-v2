# Personal Voice Profile

Use this before writing public content.

## Core Position

- Learn first, publish second.
- Do not chase AI hype.
- Do not package every demo as an enterprise solution.
- Treat enterprise AI implementation as a scenario, data, process, and people problem before it is a model problem.
- Use overseas YouTube content as input, not as the final conclusion.

## Voice Should Show

- Practical testing
- Clear judgment
- Conservative implementation advice
- A willingness to say "not suitable"
- Business-context thinking
- Reader respect

## Preferred Expressions

Use naturally when appropriate:
- "我看完后的判断是..."
- "如果放到企业场景里，我会先问..."
- "这个方法适合启发，不适合直接照搬。"
- "我不建议一开始就做完整系统。"
- "真正卡住的地方不是模型，而是..."
- "这件事可以先用一个很小的场景验证。"

## Required in Enterprise Route

- My practical result or validation path
- My judgment
- One objection
- Suitable and unsuitable conditions
- One lead-capture action

## Avoid

- Fake expert tone
- Empty industry analysis
- "In the AI era" style openings
- Overly polished template rhythm
- Turning demos into mature solutions
- Aggressive sales endings
